# Contributing to University of South Dundee Website

Thank you for your interest in contributing to our website! Here are some guidelines to help you get started.

## Code Style
- Use semantic HTML5 elements
- Follow BEM methodology for CSS
- Use meaningful variable names in JavaScript
- Comment your code where necessary

## Testing
- Test on multiple browsers (Chrome, Firefox, Safari, Edge)
- Test on mobile devices and tablets
- Ensure accessibility standards are met
- Verify all links work correctly

## Pull Request Process
1. Ensure your code follows our style guidelines
2. Update documentation if needed
3. Test thoroughly on different devices
4. Provide clear description of changes
5. Wait for review from maintainers
